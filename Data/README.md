# Data

This folder contains example datasets and reference files used in the TRUHiC study.

## Files

| File | Description |
|------|-------------|
| `GM12878_raw.zip.001` | Part 1 of the compressed GM12878 raw Hi-C dataset. |
| `GM12878_raw.zip.002` | Part 2 of the compressed GM12878 raw Hi-C dataset. |
| `chromosome.txt` | List of chromosomes used in the analyses. |
| `mm9.chrom.sizes.txt` | Mouse mm9 chromosome size reference file. |
| `data_extraction_via_7zip_tool.txt` | Instructions for extracting multi-part compressed archives using 7-Zip. |

## Notes

The GM12878 dataset is distributed as a multi-part archive. Download all archive parts before extraction.

For extraction instructions, please refer to `data_extraction_via_7zip_tool.txt`.

Please refer to the main repository README for additional information on data sources and preprocessing workflows.
